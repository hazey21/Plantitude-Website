2.2 Minimum Requirement Features

2.2.1 HTML5

  1. Appropriate HTML5 tags

    Throughout the home page correct HTML5 tags were used such as: header, footer, nav, and head to name a few.

  2. Inline images both static and clickable

    Static images were used on the join the team page and a clickable image is used on the Plantitude logo at the top of the page that links to the home page.

  3. Use of appropriate special characters

    Special characters are used in the footer to highlight the text for example a telephone special character is used beside the phone number.

  4. Use of appropriate meta tags

    Meta charset tells the browser what character encoding to use (UTF-8), as well as meta keywords, viewport, content, and name.

  5. Footer on home page indicating when website was last updated or modified

    The footer on the home page features a last update funtion using document.lastModified.

2.2.2 CSS3

  1. Use CSS3 to format appearance of paragraphs, lists, headers, background image, tables and forms

    -In CSS I customized the paragraph at the bottom of the Join the Team page using background-colors, larger text, borders, font-style, and font-color.

    -Using CSS I enhanced an unordered list to create a navigation menu with margins, paddings, background-colors, postioning, and display to create the finished product.

    -I also used the header to help contain the navigation menu and logo design.

    -On the home page near the bottom I inserted a background image using background-size to control its width and height.

    -On the location page I developed a table to showcase the opening hours for the restaurant.

    -On the reserve and takeout tab there is a custom form for customers to fill out when making reservations to the restuarant.

  2. Use CSS3 contextual selectors (e1, e2, e3; e f; e > f; e + f) and at least two attribute selectors excluding class and id attribute selectors.

    I used three elements in developing the navicon menu for mobile design by selecting them into displaying as a block.

    Two elements (nav ul) were used in creating the navigation menu by having both list style, margins and paddings changed.

    The reservation form used e > f to select direct children for designing the flex layout.

    I used e + e in the join the team section to add adjacent siblings to show opacity throughout all three images.

    The two other attrbitue selectors I used were in the creation of the reservation form for example: placeholders, min and max for date/ time, as well as using type text for customizing the text boxes.

  3. Use CSS3 styles for color (RGBA& HSLA)

    The background color of the wedding form was created using RGBA and the hovered color on the navbar uses HSLA

  4. Use of dynamic pseudo-classes to highlight hyperlinks states of link, visited, active, hover; focus and two structural pseudo-classes: first-of-type and last-of-type

    The visited pseudo-class is used on the home page wedding catering section to have the color changed once clicked.

    The active pseudo-class is used on the home page location section to have the link colored black once activated by the user.

    The hover pseudo-class is used on the navigation menu to highlight a different color when users are hovering over the text.

    The focus pseudo-class is used on the reservation form to change the font-size and border when activated.

    The first of letter is used to customize the lettering at the start of all the subheadings on the menu.

    The last of type was used to change the text coloring in the last paragraph of the join the team page.

  5. Use of CSS3 background styles (background size, background-clip, background-origin, padding, margins, borders, use of an appropriate border style, use of rounded corners) -

    Background size is set to cover on the home page with the wedding/ reserve portion.

    Background-clip is set to initial in the reservation form.

    Background-orgin starts from the border of the reservation form.

    Paddding is used on the navigation menu to seperate the different tabs.

    Margins are used to center the Plantitude logo on the home page.

    A border was made around the wedding reservation form.

    The border style groove was used around the wedding reservation form to give it an appealing look.

    Rounded- corners were also used around the wedding reservation form.

  6. Your home page should have a drop cap created with CSS3

    Drop-cap was created at the start of all the main headings on both the dinner and brunch menu.

  7. Define document content with the class and id properties and create styles for them

    8. The class attribute was used in the open hours table to group all the elements together needed to customize and position it properly.

    The id attribute was used to select the wedding form and customize its background-colors, box shadows and other elements accordingly. Only 1 unqiue value is allowed by definition of id.

  8. Use each of the following page layout techniques: Grid, Flex, Float, Positioning

    The grid layout technique was used in formating the home page.

    The flex layout technique was used in formating the reserve page.

    The float layout technique was used in formating the brunch and dinner menu page.

    The positioning layout technique was used in formating the wedding catering page.

  9. Use of CSS3 text and box shadows

    Customized text was used on the menu page including: font-size, line-spacing, and font-family.

    Box shadows were used on the wedding form to make it stand out to the user.

  10. Use of CSS3 transform rotate feature

    The transform rotate element was used at the top of the menu where I rotated the "choice of multigrain" by 1deg.

  11. Use of CSS3 opacity applied to two images to appear semi-transparent

    Opacity was used on three images at the top on the join the team page.
